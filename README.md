=============

# プロジェクトの概要

市販のスピードガンを改造し、球速をカメラで読み取り、記録をNFTで発行します。

下記の技術を利用しています。
* スピードガン用マウントベースの作成は3D Printer
* 球速の文字認識にはTensorFlow
* 音声の読み上げにはVOICEVOX
* NFTの発行にはSymbolブロックチェーン

使用したスピードガンはこちら  
Bushnell 101911 Velocity Speed Gun
